СЕКЦИИ
Список секций:
Создание: http://localhost/sections
Отправляемые данные:
name: "Бокс"
Удаление: http://localhost/sections/{id}
ТРЕНЕРА
Список тренеров:
Создание: http://localhost/trainers
Отправляемые данные:
username: "Иванов Иван Иванович", password: "password", email: "example@example.com", phoneNumber: "+79999999999"
Удаление: http://localhost/trainers/{id}
Установка почасовой ставки: http://localhost/trainers/{id}/set-hourly-rate
Отправляемые данные:
hourlyRate: 500
ТРЕНИРОВКИ
Список тренировок:
Создание: http://localhost/trainers/{id}/trainings
Отправляемые данные:
startTime: "2024-10-03T07:29:48", endTime: "2024-10-03T07:29:48"
Удаление: http://localhost/trainers/trainigs/{id}
КЛИЕНТЫ
Список клиентов:

username: "Иванов Иван Иванович", password: "password", email: "example@example.com", phoneNumber: "+79999999999"

Абонементы
Список абонементов:

duration: 30

АДМИНЫ
Список админов:
Создание: http://localhost/admins
Отправляемые данные:
username: "Иванов Иван Иванович", password: "password", email: "example@example.com"
Удаление: http://localhost/admins/{id}
Получить проведенные тренировки:
http://localhost/admins/trainings-conducted

count: Количество проведенных тренировок, trainings: Список проведенных тренировок

http://localhost/admins/clients-activity
Получить активность тренеров:
http://localhost/admins/trainers-activity
